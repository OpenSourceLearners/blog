常用命令

  • git add . # 将当前所有文件提交到暂存区，也可以一个或多个文件
    • git add -A # 提交所有变化
    • git add -u # 提交被修改(modified)和被删除(deleted)文件，不包括新文件(new)
    • git add . # 提交新文件(new)和被修改(modified)文件，不包括被删除(deleted)文件
  • git reset a.html # 去除暂存区的a.html文件
  • git checkout a.html # 把a.html文件还原到初始状态
  • git commit -m update # 把暂存区的文件提交到本地仓库，update是备注
  • git pull origin master # 更新本地仓库
  • git push origin master # 将本地提交推送到远程仓库
  • git branch fature/login # 创建分支fature/login
  • git checkout master # 切换分支master
  • git log # 查看提交日志
  • git rm file.html # 从版本库中删除file.html文件
  • git remote add origin git@ github:robbin/robbin_site.git # 添加远程仓库地址
